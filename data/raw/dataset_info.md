# Dataset

The raw CSV files (FIFA 15–22) are not stored in this 
repository due to file size limits.

**Source:** FIFA 22 Complete Player Dataset — Kaggle  
**Link:** https://www.kaggle.com/datasets/stefanoleone992/fifa-22-complete-player-dataset

The dataset was downloaded from Kaggle and hosted on GitHub. 
Azure Data Factory then ingested the CSV files from GitHub 
into Azure Data Lake Storage Gen2 using HTTP linked services.
